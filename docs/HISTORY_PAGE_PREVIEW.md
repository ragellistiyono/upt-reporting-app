## Performance Considerations

### Data Fetching
- âœ… Fetches only user's own submissions via `Query.equal('submitted_by_user', user.$id)`
- âœ… Ordered by creation date (newest first) via `Query.orderDesc('$createdAt')`
- âœ… Cached by React state (no re-fetch on re-render)
- âœ… Loading state prevents layout shift

### Rendering Optimization
- âœ… Conditional rendering reduces unnecessary DOM nodes
- âœ… Map with stable keys (using `submission.$id`)
- âœ… Truncated titles prevent excessive cell width
- âœ… Hover effects use CSS (no JavaScript)

### Accessibility
- âœ… Semantic HTML reduces screen reader processing
- âœ… Title attributes provide full text without expanding cells
- âœ… Links open in new tab (preserves user session)
